## 概要

AWS CDK へコントリビュートするまでの流れを実際に体験できるワークショップコンテンツとなります。

## 前提条件

- Admin 相当の権限が使える AWS アカウントを持っていること
- 本ワークショップで使用できる GitHub アカウントを持っていること
- 上記 AWS アカウントで最大 3US ドル程度の課金額の発生にご理解いただけること

## ワークショップの流れ

左のバーに表示された順番に沿って着手していきます。

まずは、実際にコントリビュートを体験するための開発環境を、AWS アカウント上の EC2 に構築します。

[環境構築](/cdk-conf-2024-contribute-workshop/1-環境構築/setup)ページへ飛んでみましょう。

## ワークショップ終了時の注意

本ワークショップで作成した開発環境の EC2、またもしワークショップ中に作成した AWS リソースがあれば、放置すると料金がかかってしまうものがあります。

そのため、本ワークショップ終了時にはそれらのリソースは全て削除しましょう。

詳細は、最後の[環境削除](/cdk-conf-2024-contribute-workshop/6-環境削除/env-deletion)ページをご覧下さい。
